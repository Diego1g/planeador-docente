# `.lesson`

This folder contains the **supporting materials** used to generate lessons.

## Contents

- `templates/`: base templates used by scripts to create lesson artifacts:
	- `HIGHLIGHTS.md`
	- `DISCOURSE.md`
	- `SLIDES.md`
	- `.lesson-config.json`
- `assets/`: visual resources for the workflow:
	- `workflow.dot` (diagram source)
	- `workflow.png` (diagram image)
