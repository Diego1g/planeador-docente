# Templates

This folder contains structure baselines for lesson artifacts.

## Available Templates

- `.lesson-config.json` — course metadata and lesson list schema
- `HIGHLIGHTS.md` — co-teacher discussion summary structure
- `DISCOURSE.md` — teaching narrative and slide handoff structure
- `SLIDES.md` — MARP deck structure baseline

## Usage Notes

- Templates are guides, not fixed outputs.
- Always respect lesson language requirements.
- Always respect the decided lesson section count (do not force example counts in templates).

## Typical Flow

1. Plan with `@course-planner`
2. Generate highlights with `@discussion-moderator`
3. Generate discourse (optional) with `@lesson-planner`
4. Generate slides with `@slides-maker`
